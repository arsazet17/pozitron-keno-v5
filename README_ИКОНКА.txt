ПОЗИТРОН КЕНО v5 — иконка как в v3

Загрузить в корень репозитория:
- index.html
- manifest.webmanifest
- папку assets целиком:
  - icon.svg
  - icon-192.png
  - icon-512.png

После публикации:
1. Удалить старый ярлык КЕНО v5 с рабочего стола.
2. Открыть сайт заново.
3. В меню браузера выбрать «Добавить на главный экран» / «Установить приложение».
Тогда появится новая иконка-мишень.
