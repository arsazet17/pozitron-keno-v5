ЗАГРУЗКА НА GITHUB С ТЕЛЕФОНА:
1. Откройте репозиторий pozitron-keno-v4.
2. Нажмите три точки рядом с Code.
3. Нажмите Upload file.
4. Выберите ВСЕ четыре файла из этой папки.
5. Внизу нажмите Commit changes.
